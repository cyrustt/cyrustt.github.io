# Cyrus Tsui — Photography Portfolio

[View the live portfolio](https://cyrustt.github.io/)

A responsive photography portfolio I designed and built in 2022 to present portrait, landscape, and product work.

## Highlights

- Responsive image galleries for multiple photography categories
- Dedicated portfolio, about, and contact pages
- Custom layout and typography built on Bootstrap
- Static deployment through GitHub Pages

## Built with

- HTML
- CSS
- JavaScript
- Bootstrap 5
- GitHub Pages

## Run locally

No build step is required. Clone the repository and serve the static files:

```bash
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).
